# entities (domain.model.entities)

Propósito técnico

Contiene entidades del dominio que poseen identidad (por ejemplo `Category`). Estas entidades pueden formar parte de aggregates o ser referenciadas por ellos.

Qué parte del enunciado resuelve

- Implementa la entidad `Category` y su mapeo a la tabla `categories` pre-poblada.

Archivos que deberían ir aquí

- `Category.java`.

Significado en DDD (técnico)

Una entidad es un objeto con identidad persistente que representa un concepto del negocio.

Explicación para no programadores

Una entidad es un registro con identidad (como una categoría: Tractors, Harvesters) que sirve para clasificar las máquinas.
